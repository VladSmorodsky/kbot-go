# kbot-go
Golang Telegram Bot. It shows capital city of entered country.

# Usage
Go to telegram and click on send message button: [KBOT-GO](https://t.me/dk_go_bot)

# Commands
- Get version:
```
/hello
```
- Get capital city of country
```
/capital <country>
```
Example: 
``` 
/capital usa 
```
Result:
```
The capital of USA is Washington, D.C.
```
